# 🍽 Python Restaurant Ordering System

A beginner-friendly Python mini project that simulates a simple restaurant ordering and billing system using basic programming concepts.

---

## 📌 Project Description

This project allows users to order food items from a predefined menu and calculates the total bill amount based on their selections. It demonstrates how real-world restaurant billing logic can be implemented using Python.

---

## 🚀 Features

- Displays restaurant menu with prices  
- Takes user input for food ordering  
- Adds selected items to total bill  
- Option to add another item  
- Displays final total bill  

---

## 🛠 Concepts Used

- Python Dictionary  
- If-Else Conditions  
- User Input Handling  
- Variables  
- Basic Arithmetic Operations
